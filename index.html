Descripción del Programa

El programa de Ingeniería de Software de UCompensar está diseñado para formar profesionales con habilidades técnicas y de gestión para el desarrollo de software de alta calidad. Los estudiantes aprenderán sobre programación, bases de datos, ingeniería de requisitos, diseño de software, pruebas y aseguramiento de la calidad, entre otros temas.

Plan de Estudios

El plan de estudios está compuesto por una serie de asignaturas que cubren tanto los fundamentos teóricos como las aplicaciones prácticas de la ingeniería de software. A continuación, se presenta una lista de algunas de las asignaturas que se ofrecen:

  • Introducción a la Programación
  • Estructuras de Datos
  • Ingeniería de Requisitos
  • Diseño de Software
  • Pruebas y Aseguramiento de la Calidad
  • Gestión de Proyectos de Software
  • Arquitectura de Software
  • Desarrollo de Aplicaciones Web
  • Desarrollo de Aplicaciones Móviles
  • Inteligencia Artificial
  • Seguridad Informática
  • Base de Datos
  • Redes de Computadoras
  • Computación en la Nube
  • Ética Profesional
